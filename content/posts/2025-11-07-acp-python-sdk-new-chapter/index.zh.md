当我最初开始编写 **[Agent Client Protocol Python SDK](https://github.com/agentclientprotocol/python-sdk)** 时，它只是一个出于好奇的小项目。
我想弄清楚 *coding agents* 是如何交流的——状态、意图与消息在客户端和智能体之间如何流动。

随着时间推移，它逐渐成长为一个真正有用的项目。
越来越多的开发者开始使用它，从 **Moonshot** 到其他探索智能体工作流的公司。
SDK 的架构变得稳定，并持续与 ACP 规范保持同步，成为多种实际集成的基础。

今天，是一个新的开始。
Python SDK 已经正式迁移到 **[agentclientprotocol](https://github.com/agentclientprotocol)** 组织下，与 TypeScript、Rust、Kotlin SDK 一起成为官方生态的一部分。

我会继续在新组织下维护和改进它，专注于让它更加可靠、简洁、易于集成，让更多开发者能够轻松构建智能体系统。

特别感谢 **Sergey Ignatov**、**Anna** 和 **Ben** 在整个迁移过程中给予的信任与支持，也感谢所有曾经使用、贡献或反馈过这个项目的开发者们。

这个项目最初只是一个实验，如今它已成为一个共同的努力。
愿我们一起迎接新的篇章。

— **尚卓燃（PsiACE）**
Maintainer, ACP Python SDK
